zlmkeygen
=========

`zlmkeygen` is a minimal key pair generation tool for ZLM2. Install with:

    go get -v github.com/wikena/zlmkeygen

Afterwards run:

    zlmkeygen > /path/to/vendor.zlmlic

And send the `PublicKey` contained in `/path/to/vendor.zlmlic` to us.
